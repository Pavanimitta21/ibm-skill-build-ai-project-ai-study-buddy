# AI Study Buddy 📚

## Description
An AI-powered study assistant that explains topics, provides summaries, and generates quizzes.

## Features
- Topic explanation
- Summary generation
- Interactive MCQ quiz
- Score evaluation

## Setup Instructions
1. Install dependencies:
   pip install -r requirements.txt

2. Add your OpenAI API key in app.py

3. Run the app:
   streamlit run app.py
